## First-Time Setup

### 1. Login to Super Admin

Visit `https://super-admin.yourdomain.com/login`

Use the admin account created during installation.

### 2. Create First Tenant

1. Go to **Tenant Management** â†’ **Add Tenant**
2. Fill in tenant information:
   - Company name
   - Contact email
   - Subdomain (e.g., `store1`)
3. Click **Create**

### 3. Configure Tenant

1. Enter tenant details
2. Configure:
   - Domain binding
   - Payment methods
   - Email notifications

### 4. Tenant Login

Tenant admin visits `https://admin.yourdomain.com/login`

Login with assigned credentials to:
- Add products
- Set categories
- Configure themes
