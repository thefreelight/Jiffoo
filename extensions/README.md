# Extensions Placeholder

This directory is intentionally kept as a placeholder in the public repository.

## Why it is empty

Official themes and plugins are **not** bundled into the open-source repo. Self-hosted
installations download official marketplace packages from Jiffoo Platform after the
instance is connected and the store is bound.

## What lives here at runtime

- `extensions/themes/` stores downloaded theme packages
- `extensions/plugins/` stores downloaded plugin packages

These runtime assets are generated after install and are excluded from version control.
