# App Landing Page Theme

Official app download landing page theme for Android APK distribution.

## Direction

- High-confidence download flow with direct APK CTA and QR scan support
- Real app screenshot as the first-viewport visual anchor
- Version, package, target SDK, and checksum details visible near the CTA
- Mobile-first layout for users arriving from Android devices

## Visual System

- Ink: `#0b1220`
- Surface: `#f7faff`
- Primary blue: `#176bff`
- Deep blue: `#0b4edb`
- Heading font: `Manrope`
- Body font: `Manrope`
