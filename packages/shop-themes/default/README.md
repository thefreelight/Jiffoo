# Jiffoo Mall 默认主题

这是 Jiffoo Mall 的默认主题包，为所有租户提供开箱即用的功能性 UI。

## 特性

- 响应式设计
- 可访问性支持
- 使用 Tailwind CSS 和 CSS 变量
- 支持主题配置自定义

## 组件

- `HomePage` - 首页
- `ProductsPage` - 商品列表页
- `ProductDetailPage` - 商品详情页
- `CartPage` - 购物车页
- `CheckoutPage` - 结账页
- `NotFound` - 404 页面

## 开发

```bash
# 安装依赖
pnpm install

# 类型检查
pnpm type-check
```

## 使用

主题包会被前端应用自动导入和使用，无需手动引用。
