# Jiffoo Mall Default Theme

This is the default theme package for Jiffoo Mall, providing out-of-the-box functional UI for all tenants.

## Features

- Responsive design
- Accessibility support
- Uses Tailwind CSS and CSS variables
- Supports theme configuration customization

## Components

- `HomePage` - Home page
- `ProductsPage` - Product list page
- `ProductDetailPage` - Product detail page
- `CartPage` - Shopping cart page
- `CheckoutPage` - Checkout page
- `NotFound` - 404 page

## Development

```bash
# Install dependencies
pnpm install

# Type check
pnpm type-check
```

## Usage

The theme package is automatically imported and used by the frontend application, no manual reference needed.
