# Digital Vault Theme

Digital Vault is an embedded shop theme for instant-delivery virtual goods:

- gift cards
- redemption codes
- account credentials
- download assets
- license keys

It is currently being rebuilt as Jiffoo's Dujiao-inspired digital-goods storefront reference:

- shopper flow benchmark: `dujiao-next/user`
- operator/admin benchmark: `dujiao-next/admin`
- reusable digital-goods business domains are expected to move into plugins or backend modules rather than staying theme-owned

The theme should feel like a production digital-goods store, not a generic marketing shell with digital copy pasted on top.
