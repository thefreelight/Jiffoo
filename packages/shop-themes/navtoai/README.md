# NavToAI Theme

NavToAI is an embedded storefront theme for AI tool directories and curated workflow catalogs:

- AI chat and search products
- image, video, and audio creation tools
- coding, agent, and automation stacks
- editorial-style discovery pages
- commerce-ready checkout and account flows

It is designed for operators who want a calmer, more curated navigation-site presentation while still keeping Jiffoo's storefront and ordering flows available.
