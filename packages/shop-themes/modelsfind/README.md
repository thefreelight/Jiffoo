# ModelsFind Theme

ModelsFind is a premium editorial storefront theme for model directories, private booking flows, and AI-assisted shortlist discovery.

## Focus

- nocturne-inspired visual direction
- coordinated desktop and mobile layouts
- booking-first product and order flows
- curated account, concierge, and support surfaces

## Page Coverage

Custom `modelsfind` implementations currently cover:

- home, products, product detail
- categories, search, deals, bestsellers, new arrivals
- cart, checkout, orders, order detail, order success, order cancelled
- login, register, auth callback
- profile, profile settings
- help, contact, privacy, terms, not found

`Toast` still reuses the default storefront implementation.
