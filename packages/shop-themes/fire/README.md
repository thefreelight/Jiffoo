# Fire Theme

Fire is the official finance-first storefront theme pack for Jiffoo.

It is designed as a lightweight FIRE dashboard shell:

- the theme owns the visual language, layout, and theme-pack contract
- the upload-driven finance agent block lives in the runtime host inside `jiffoo-mall-core`
- the goal is a fast MVP for mixed personal and company balance-sheet review

## Product intent

The storefront should feel calm, operational, and trustworthy.

Primary experiences:

- upload statements, CSV exports, screenshots, and plain-text ledgers
- separate personal debt from company leverage
- surface liquidity, strategic assets, and estimated FIRE trajectory

## Packaging scope

This repository keeps the downloadable theme-pack source only:

- `theme-pack/theme.json`
- `theme-pack/tokens.css`
- `theme-pack/templates/*`
- `theme-pack/schemas/*`
