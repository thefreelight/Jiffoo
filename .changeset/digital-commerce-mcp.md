"create-jiffoo-app": minor

Added vertical template system with `--template` flag. New templates: `digital-goods` (Digital Vault theme with gift cards, codes, licenses) and `esim` (eSIM Mall theme with travel data plans). Templates include pre-configured themes, seed datasets, and environment presets for instant digital commerce setup.

---
"@jiffoo/mcp-server": minor
---

New package: MCP (Model Context Protocol) Server for Jiffoo e-commerce. Enables AI agents (Claude Desktop, Claude Code) to browse products, manage carts, and create orders. Supports stdio and HTTP transports. Includes scoped API token authentication system.
