# 快速开始

## 概述

Jiffoo Mall 是一个多租户电商平台，支持：
- 🏪 多商户入驻
- 🎨 主题定制
- 🔌 插件扩展
- 👥 三级代理分销

## 系统角色

| 角色 | 说明 | 访问地址 |
|------|------|----------|
| 超级管理员 | 平台运营方 | `/admin` |
| 商户管理员 | 入驻商家 | `/tenant` |
| 代理商 | 分销代理 | `/agent` |
| 消费者 | 购物用户 | `/shop` |

## 安装后首次使用

### 1. 登录超级管理后台

访问 `https://admin.yourdomain.com/login`

使用安装时创建的管理员账号登录。

### 2. 创建第一个商户

1. 进入 **租户管理** → **添加租户**
2. 填写商户信息：
   - 公司名称
   - 联系邮箱
   - 子域名 (如 `store1`)
3. 点击 **创建**

### 3. 配置商户

1. 进入商户详情
2. 设置：
   - 域名绑定
   - 支付方式
   - 邮件通知

### 4. 商户登录

商户管理员访问 `https://tenant.yourdomain.com/login`

使用分配的账号登录，开始：
- 添加商品
- 设置分类
- 配置主题

## 核心功能

### 商品管理

```
商户后台 → 商品管理 → 添加商品
```

支持：
- 多规格 SKU
- 库存管理
- 价格设置
- 商品分类

### 订单管理

```
商户后台 → 订单管理
```

功能：
- 订单列表
- 发货处理
- 退款管理
- 订单导出

### 插件系统

```
商户后台 → 插件商城
```

可安装：
- 支付插件 (Stripe)
- 邮件插件 (Resend)
- 登录插件 (Google OAuth)
- 分销插件 (Affiliate)

### 主题定制

```
商户后台 → 主题设置
```

支持：
- 主题切换
- 颜色定制
- Logo 上传
- 布局调整
