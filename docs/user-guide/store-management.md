# 商城管理指南

## 商品管理

### 添加商品

1. 进入 **商品管理** → **添加商品**
2. 填写基本信息：
   - 商品名称
   - 商品描述
   - 商品分类
   - 商品图片

3. 设置价格和库存：
   - 基础价格
   - 库存数量
   - SKU 编码

4. 配置规格 (可选)：
   - 添加规格项 (如颜色、尺寸)
   - 设置规格值
   - 配置各规格组合的价格和库存

5. 点击 **发布**

### 商品分类

```
商品管理 → 分类管理
```

- 支持多级分类
- 可设置分类图标
- 支持排序

### 库存管理

```
商品管理 → 库存管理
```

功能：
- 批量调整库存
- 库存预警设置
- 库存变动记录

## 订单管理

### 订单列表

```
订单管理 → 全部订单
```

筛选条件：
- 订单状态
- 下单时间
- 支付状态
- 客户信息

### 订单处理流程

```
待付款 → 待发货 → 已发货 → 已完成
           ↓
        已取消/已退款
```

### 发货操作

1. 选择待发货订单
2. 点击 **发货**
3. 填写物流信息：
   - 物流公司
   - 运单号
4. 确认发货

### 退款处理

1. 进入订单详情
2. 点击 **退款**
3. 选择退款类型：
   - 仅退款
   - 退货退款
4. 填写退款金额
5. 确认退款

## 客户管理

### 客户列表

```
客户管理 → 客户列表
```

查看：
- 注册信息
- 订单历史
- 消费统计

### 客户分组

支持按条件自动分组：
- 消费金额
- 订单数量
- 注册时间

## 数据统计

### 销售概览

```
数据中心 → 销售概览
```

指标：
- 今日销售额
- 订单数量
- 客单价
- 转化率

### 商品分析

```
数据中心 → 商品分析
```

- 热销商品排行
- 商品浏览量
- 加购转化率

### 客户分析

```
数据中心 → 客户分析
```

- 新增客户
- 复购率
- 客户来源

## 设置

### 店铺设置

```
设置 → 店铺设置
```

- 店铺名称
- 店铺 Logo
- 联系方式
- 营业时间

### 支付设置

```
设置 → 支付设置
```

配置支付方式：
- Stripe
- PayPal
- 微信支付
- 支付宝

### 物流设置

```
设置 → 物流设置
```

- 运费模板
- 物流公司
- 发货地址

### 通知设置

```
设置 → 通知设置
```

配置通知：
- 订单通知
- 发货通知
- 营销通知
