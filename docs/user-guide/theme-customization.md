# 主题定制指南

## 主题商城

访问路径：`商户后台 → 主题商城`

### 浏览主题

主题分类：
- 🛍️ 通用商城
- 👗 时尚服饰
- 🍔 餐饮外卖
- 💻 数码科技
- 🏠 家居生活

### 安装主题

1. 选择喜欢的主题
2. 预览主题效果
3. 点击 **安装** 或 **购买**
4. 完成支付 (付费主题)
5. 主题添加到库中

## 主题设置

### 切换主题

```
主题设置 → 我的主题
```

1. 选择已安装的主题
2. 点击 **应用**
3. 确认切换

### 基础配置

```
主题设置 → 基础配置
```

**Logo 设置：**
- 上传 Logo 图片
- 推荐尺寸: 200x60px
- 支持 PNG/SVG

**Favicon：**
- 上传网站图标
- 推荐尺寸: 32x32px

**店铺名称：**
- 显示在页面标题
- 显示在 Logo 旁边

### 颜色定制

```
主题设置 → 颜色
```

可定制颜色：
| 颜色项 | 说明 | 默认值 |
|--------|------|--------|
| 主色调 | 按钮、链接 | #3B82F6 |
| 辅助色 | 次要元素 | #10B981 |
| 背景色 | 页面背景 | #F9FAFB |
| 文字色 | 正文文字 | #111827 |
| 边框色 | 边框线条 | #E5E7EB |

**使用颜色选择器：**
1. 点击颜色块
2. 选择颜色或输入色值
3. 实时预览效果
4. 点击 **保存**

### 字体设置

```
主题设置 → 字体
```

可选字体：
- 系统默认
- Inter
- Roboto
- Noto Sans SC (中文)
- 自定义字体

字体大小：
- 标题字号
- 正文字号
- 小字号

### 布局设置

```
主题设置 → 布局
```

**导航栏：**
- 位置: 顶部固定 / 顶部滚动
- 样式: 透明 / 纯色
- 菜单项配置

**首页布局：**
- Banner 轮播
- 商品推荐
- 分类展示
- 促销区域

**商品列表：**
- 每行显示数量
- 列表/网格切换
- 排序选项

**商品详情：**
- 图片展示方式
- 规格选择样式
- 相关推荐

### 页脚设置

```
主题设置 → 页脚
```

配置项：
- 版权信息
- 联系方式
- 社交媒体链接
- 快速链接

## 高级定制

### 自定义 CSS

```
主题设置 → 高级 → 自定义 CSS
```

添加自定义样式：
```css
/* 示例：修改按钮圆角 */
.btn-primary {
  border-radius: 9999px;
}

/* 示例：修改卡片阴影 */
.product-card {
  box-shadow: 0 4px 6px rgba(0, 0, 0, 0.1);
}
```

### 自定义代码

```
主题设置 → 高级 → 自定义代码
```

可添加：
- 头部代码 (统计代码)
- 底部代码 (客服插件)

## 主题预览

### 实时预览

修改设置时可实时预览效果：
1. 修改配置项
2. 右侧预览区自动更新
3. 满意后点击 **保存**

### 设备预览

支持预览不同设备效果：
- 🖥️ 桌面端
- 📱 平板端
- 📱 手机端

## 主题备份

### 导出配置

```
主题设置 → 备份 → 导出
```

导出当前主题配置为 JSON 文件。

### 导入配置

```
主题设置 → 备份 → 导入
```

从 JSON 文件恢复主题配置。

## 常见问题

**Q: 切换主题会丢失配置吗？**
A: 不会，每个主题的配置独立保存。

**Q: 如何恢复默认设置？**
A: 主题设置 → 重置为默认

**Q: 自定义 CSS 不生效？**
A: 检查 CSS 语法，尝试添加 `!important`
