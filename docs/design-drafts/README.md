# Jiffoo Mall 设计草稿

这个目录存放设计风格的草稿和模板，不会影响主项目代码。

## 目录结构

```
docs/design-drafts/
├── README.md                    # 本文件
└── templates/
    └── all-styles-preview.tsx   # 所有风格预览组件（完整版）
```

## 设计风格清单

### 1. Stellar 科技风 ⭐ (推荐用于模板市场)
- 深色背景 (#0B0F1A)
- 星空粒子动画
- 紫色/蓝色渐变光晕
- 玻璃拟态卡片
- 适合：SaaS、科技产品、高端电商

### 2. 现代简约风 ✅ (推荐作为基础模板)
- Apple/Stripe 风格
- 蓝色主色调
- 干净的白色背景
- 适合：通用电商、品牌官网

### 3. 渐变霓虹风
- 鲜艳渐变 + 发光效果
- 深色背景
- 适合：潮牌、年轻化品牌

### 4. 毛玻璃未来风
- 半透明 + 模糊背景
- 彩色渐变底色
- 适合：创意产品、艺术品

### 5. 暗黑奢华风
- 深色 + 金色点缀
- 高端质感
- 适合：奢侈品、高端定制

### 6. 流体动态风
- 液态动画 + 波浪效果
- 3D 倾斜卡片
- 适合：游戏、娱乐产品

## 使用方式

这些草稿可以：
1. 作为模板市场的模板参考
2. 提取组件到正式项目中
3. 作为设计规范的参考
